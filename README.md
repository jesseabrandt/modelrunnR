# modelrunnR

<!-- badges: start -->
<!-- badges: end -->

An R package for orchestrating R and Python model runs through a common harness
interface and lightweight configuration. In early development.

## Installation

You can install the development version of modelrunnR from
[GitHub](https://github.com/jesseabrandt/modelrunnR) with:

``` r
# install.packages("pak")
pak::pak("jesseabrandt/modelrunnR")
```
